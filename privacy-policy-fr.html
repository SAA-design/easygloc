Politique de Confidentialité pour EasyGluco

Dernière mise à jour : Février 2025

Introduction

Chez EasyGluco, nous priorisons votre confidentialité dès la conception. Notre application fonctionne entièrement sur votre appareil, sans collecte de données ni communication avec des serveurs externes.

Stockage Local des Données

Toutes vos données, notamment :

  • Les mesures de glycémie
  • Les calculs d'HbA1c
  • Les paramètres personnels
  • Les préférences de rappel

sont uniquement stockées sur votre appareil et ne le quittent jamais.

Aucune Collecte de Données

Nous ne :

  • Collectons aucune information personnelle
  • Suivons pas votre utilisation
  • Utilisons pas de services d'analyse
  • Stockons pas de données sur des serveurs externes
  • Partageons aucune information avec des tiers

Intégration avec Apple Health

Si vous choisissez d'utiliser l'intégration avec Apple Health, cette interaction est gérée directement par le système d'exploitation de votre appareil. Nous ne recevons ni ne traitons aucune de ces données.

Contrôle de Vos Données

Vous avez un contrôle total sur vos données. Vous pouvez :

  • Supprimer toutes les données de l'application à tout moment
  • Exporter vos données pour un usage personnel
  • Gérer les autorisations de l'application via les paramètres de votre appareil
